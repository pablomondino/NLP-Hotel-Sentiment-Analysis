# Análisis de Sentimiento en el Sector Hotelero - Punta Cana 🌴🤖

Este proyecto forma parte de mi certificación en **Data Science III**. En él, desarrollo un modelo de **Procesamiento de Lenguaje Natural (NLP)** aplicado a la experiencia del cliente en hoteles de la República Dominicana, utilizando Redes Neuronales Recurrentes (RNN).

## 📊 Resumen del Proyecto
El objetivo principal fue analizar un dataset de más de **32,000 reseñas** de hoteles para identificar patrones de satisfacción y sentimientos de los usuarios. Mediante técnicas de Deep Learning, el modelo clasifica las opiniones para facilitar la toma de decisiones basada en datos en el sector turístico.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 🐍
* **Entorno:** Google Colab 🚀
* **Librerías principales:** * `Pandas` & `NumPy` (Procesamiento de datos)
    * `NLTK` / `Spacy` (Limpieza de texto y NLP)
    * `TensorFlow` / `Keras` (Construcción de la RNN)
    * `Matplotlib` / `Seaborn` (Visualización de resultados)

## 🛣️ Metodología (User Journey)
Para este análisis, seguí un flujo de trabajo estructurado:
1.  **Ingesta de Datos:** Carga y exploración inicial del dataset.
2.  **Preprocesamiento:** Limpieza de texto (eliminación de ruidos, tokenización y lematización).
3.  **Arquitectura del Modelo:** Implementación de una Red Neuronal Recurrente para capturar la naturaleza secuencial del lenguaje.
4.  **Evaluación:** Análisis de métricas (accuracy y loss) para optimizar el rendimiento del modelo.

## 📈 Resultados
El modelo demostró una alta capacidad para categorizar experiencias de huéspedes de forma automatizada, permitiendo extraer insights valiosos sobre la calidad del servicio.
